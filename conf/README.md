# Overview

Package `conf` is a framework for reading configurations from variety of configuration stores, such as ENV, files, or Apollo.

## Supported Stores

- ENV
- Local files
- Apollo. Watching is also supported

## Supported Formats

- properties, props, prop
- json
- yaml, yml
- toml
- hcl
- tfvars
- ini
- env
- dotenv
